<h2><a href="https://www.geeksforgeeks.org/problems/boolean-parenthesization5610/1">Boolean Parenthesization</a></h2><h3>Difficulty Level : Difficulty: Hard</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18px;">You are given a boolean expression <strong>s&nbsp;</strong>containing<br>&nbsp;&nbsp;&nbsp; 'T' ---&gt; true<br>&nbsp;&nbsp;&nbsp; 'F' ---&gt; false&nbsp;<br>and following operators between symbols<br>&nbsp; &nbsp;&amp;&nbsp;&nbsp; ---&gt; boolean AND<br>&nbsp;&nbsp;&nbsp; |&nbsp;&nbsp; ---&gt; boolean OR<br>&nbsp; &nbsp;^&nbsp;&nbsp; ---&gt; boolean XOR<br>Count the number of ways we can <strong>parenthesize </strong>the expression so that the value of expression evaluates to <strong>true</strong>.</span></p>
<p><span style="font-size: 18px;">Note: The answer is guaranteed to fit within a <strong>32-bit</strong> integer.</span></p>
<p><strong><span style="font-size: 18px;">Examples:</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> s = "T|T&amp;F^T"
<strong>Output:</strong> 4
<strong>Explaination:</strong> The expression evaluates to true in 4 ways: ((T|T)&amp;(F^T)), (T|(T&amp;(F^T))), (((T|T)&amp;F)^T) and (T|((T&amp;F)^T)).</span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> s = "T^F|F"
<strong>Output:</strong> 2
<strong>Explaination:</strong> The expression evaluates to true in 2 ways: ((T^F)|F) and (T^(F|F)).</span></pre>
